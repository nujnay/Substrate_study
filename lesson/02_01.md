Substrate is a framework for building a blockchains
1. 包含哪些组件
   - 数据库 database
   - peer 2 peer network
   - consensus algorithm
   - transaction
   - runtime 状态转换函数
   - zore knowledge

2. substrate 具体的module
   - core
      - database
      - transaction queue
      - command line
      - generate privacy and public key
      - RPC remote procedure call
   - basic logic
      - data structure
      - settlement
      - timestamp
